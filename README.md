InvestDash Premium 📊
[PT]
Dashboard financeiro intuitivo desenvolvido para organizar investimentos, gerenciar gastos e planejar metas financeiras pessoais com agilidade e design moderno.

[EN]
Intuitive financial dashboard designed to organize investments, track expenses, and plan personal financial goals with agility and a modern design.

🚀 Funcionalidades Principais | Key Features
Gestão de Ativos | Asset Management: Cadastro de investimentos (Renda Variável/Fixa) com cálculo automático de preço médio. | Manage investments (Stocks/Fixed Income) with automatic average price calculation.

Controle de Despesas | Expense Tracking: Registro de gastos com categorização e visualização de movimentações recentes. | Log expenses with categorization and view recent transactions.

Simulador de Juros | Compound Interest Calculator: Cálculo de rendimentos com juros compostos para planejamento de longo prazo. | Calculate compound interest returns for long-term planning.

Metas Financeiras | Financial Goals: Barra de progresso dinâmica para acompanhar a conquista dos seus objetivos. | Dynamic progress bars to track your goals.

Persistência de Dados | Data Persistence: Uso de LocalStorage para que seus dados fiquem salvos no navegador. | Uses LocalStorage to keep your data saved in the browser.

UX Premium: Modo Claro/Escuro (Theme Switcher) e design responsivo. | Dark/Light Mode theme switcher and responsive design.

Exportação | Export: Geração de relatórios financeiros em formato CSV. | Generate financial reports in CSV format.

🛠 Tecnologias | Technologies
Front-end: HTML5, CSS3 (Variables, Flexbox, Grid), JavaScript (ES6+).

Gráficos | Charts: Chart.js para visualização interativa dos dados. | Chart.js for interactive data visualization.

Ícones | Icons: FontAwesome.

💻 Como rodar | How to run
Clone este repositório. | Clone this repository.

Abra o arquivo index.html em qualquer navegador moderno. | Open the index.html file in any modern browser.
